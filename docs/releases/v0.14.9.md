# v0.14.9 – Import: Geburtstag und höchster Rang

- XLSX-Geburtsdaten werden plausibilisiert.
- Falsche 1900-Geburtsdaten aus früheren Importläufen werden auf leer gesetzt.
- Beim erneuten Import werden plausible Geburtsdaten aus der Anmeldehistorie nachgetragen.
- Die Personenanzeige nutzt den höchsten bekannten Rang, wenn im aktiven Lagerjahr ein niedrigerer Rang gespeichert ist.
- Eine Migration schützt bestehende Lagerstatusdaten gegen Rückstufungen.
