# Release v0.14.5

## Schwerpunkt

Fix für die Personensuche und bessere Übernahme bestehender Ranginformationen.

## Änderungen

- Personensuche mit eindeutigen PDO-Platzhaltern.
- Rang-Fallback aus dem letzten bekannten Lagerstatus.
- Migration zum Nachziehen fehlender Rangangaben.
- Import aus `Zeltlager 2025 Manager.xlsx` filtert Anmeldungen auf das Quelljahr und ergänzt Rangdaten aus der Anmeldehistorie.

## Nach dem Update

1. `https://app.ritterlager.com/migration.php` ausführen.
2. Danach `public/migration.php` löschen.
3. Den Import `Zeltlager Manager 2025 importieren` erneut starten, falls Rangdaten fehlen.
