# Release v0.13.7

## Ziel

Bugfix für gebündelte Import-Quelldateien.

## Ursache

Die Importvorlagen erwarteten die Dateien exakt unter `storage/import_sources/`. Auf dem Server lagen die Dateien zwar im Projekt, aber nicht in genau diesem Pfad oder mit exakt gleicher Schreibweise. Dadurch meldete der Import „Quelldatei fehlt im Paket“.

## Änderung

`BundledImportService::sourcePath()` sucht jetzt robuster in mehreren typischen Ablageorten und vergleicht Dateinamen zusätzlich normalisiert.
