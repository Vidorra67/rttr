# Release v0.14.3

## Schwerpunkt

Ordensfarben aus dem Colorpicker werden jetzt konsistent angewendet.

## Änderungen

- `orders.color_hex` wird in weiteren Service-Abfragen geladen.
- Orden/Zelte-Karten nutzen `--order-color` für die seitliche Farbleiste.
- Mini-Badges in Dashboard, Teilnehmerliste, Personendetail, Punkteansicht und Auswertung verwenden jetzt freie Ordensfarben.
- Keine Migration erforderlich.

## Update

Dateien hochladen. Eine Migration ist für diese Version nicht notwendig. Browsercache und Service Worker bei Bedarf löschen.
