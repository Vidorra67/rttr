# Release v0.10.0

## Ziel

Auswertungen, Rangordnung, Lerneinheiten und Prüfungsergebnisse als nachvollziehbare Datenstruktur und erste Adminansichten ergänzen.

## Nicht-Ziele

- Kein vollständiger Excel-Nachbau
- Keine unkontrollierte Übernahme alter Formeln
- Kein Import aus Excel, ODS oder DOCX
- Keine automatische Endwertung ohne Prüfung

## Neu

- Auswertungsübersicht mit Zwischenstand je Orden/Zelt und Teilnehmer
- Rangstufenverwaltung
- Lerneinheitenverwaltung
- Erfassung von Prüfungsergebnissen
- Rangzuordnung je Teilnehmer
- CSV-Export der Auswertung

## Datenbank

```

```

Neue Rechte:

```text
exams.view
exams.manage
```

## Sicherheit

- Anzeigen über `exams.view`
- Bearbeiten über `exams.manage` oder `points.manage`
- CSRF bei allen Schreibaktionen
- Audit-Log für Änderungen
- Keine sensiblen Teilnehmernotizen in Auswertungen

## Update

Nach dem Einspielen ausführen:

```bash
php scripts/maintenance/migrate.php
```

Danach neu anmelden, damit neue Rechte in der Session vorhanden sind.

## Tests nach Update

- `/admin/auswertung` lädt mit aktivem Lagerjahr
- Rangstufe anlegen und bearbeiten
- Lerneinheit anlegen, bearbeiten und deaktivieren
- Prüfungsergebnis erfassen und aktualisieren
- Rang zuweisen
- CSV exportieren
- Nutzer ohne `exams.manage` kann nicht bearbeiten
- CSRF blockiert Schreibaktionen

## Bekannte Einschränkungen

- Endwertung bleibt ein fachlich zu prüfender Zwischenstand.
- Kein Import alter Excel-Wertungen.
- Keine vollständige Formelübernahme aus Excel.
