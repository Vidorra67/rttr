# Release v0.14.2

## Inhalt

Diese Version präzisiert Rangwechsel, permanente Ränge und die fachlichen Punktescopes.

## Ränge

Feste Rangfolge:

- Knappe → Ritter, 310 Punkte
- Ritter → Freiherr, 320 Punkte
- Freiherr → Graf, 330 Punkte
- Graf → Markgraf, 340 Punkte
- Markgraf → Landgraf, 345 Punkte
- Landgraf → Fürst, 350 Punkte
- Fürst → Herzog, 280 Punkte
- Herzog → Großherzog, Ernennung durch Großherzöge

Ein erreichter Rang bleibt dauerhaft erhalten und kann im Folgejahr nicht verloren werden. Das gilt auch für Herzöge, Großherzöge und Mitarbeiter.

## Punkte

- Ordnung Zelt: global pro Orden/Zelt
- Ordnung persönlich: persönlich
- Sauberkeit Geschirr: persönlich
- Prüfung: persönlich
- Spiele: global pro Orden/Zelt
- Platzdienst: global pro Orden/Zelt
- Küchendienst: global pro Orden/Zelt
- Bonus Freizeit: persönlich, max. 5 Punkte je Mitarbeiter, Teilnehmer, Tag und Freizeit-Slot

## Nach dem Update

Migration ausführen und neu anmelden. Danach `public/migration.php` wieder löschen.
