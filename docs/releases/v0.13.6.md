# Release v0.13.6

## Ziel

Bugfix für den Importfehler `SQLSTATE[HY093]: Invalid parameter number`.

## Umsetzung

In `ImportService` und `BundledImportService` wurden SQL-Abfragen korrigiert, die denselben benannten PDO-Platzhalter mehrfach in einer Query verwendet haben. Einige MySQL/PDO-Konfigurationen akzeptieren das nicht.

- `app/Services/ImportService.php`
- `app/Services/BundledImportService.php`
- `VERSION`
- `CHANGELOG.md`
- `README.md`
- `docs/releases/v0.13.6.md`
- `public/sw.js`
