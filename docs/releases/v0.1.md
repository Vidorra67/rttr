# Release v0.1.0: Projektgerüst und Baseline

## Ziel

Technische Basis für den Ritterlager Manager nach PHP-Webapp-Baseline.

## Enthalten

- Projektstruktur
- public Webroot
- Routing
- einfache Controller und Views
- zentrale Fehlerseiten
- Security-Header
- Session-Grundlage
- CSRF-Helper
- Logger
- PDO-Konfiguration
- Migration-Runner
- initiale Migrationen
- Dokumentation

## Nicht enthalten

- Login
- Rollenrechte
- Fachmodule
- Importfunktionen
- Backupfunktion
- Cronjobs mit echter Fachlogik

## Datenbank

Neue Tabellen:

- `schema_migrations`
- `app_versions`
- `settings`
- `audit_log`

## Tests nach Update

- `/` öffnet die Startseite.
- `/system/status` öffnet die Systemprüfung.
- `/unbekannt` liefert 404.
- Logs werden in `storage/logs` geschrieben.
- Migration läuft wiederholbar.
