# v0.13.1 - Public Migration Runner

## Neu

- `public/migration.php` ergänzt, um offene Migrationen einmalig per Browser auszuführen.
- Keine Loginpflicht und kein Token, damit die Datei in einfachen Hosting-Umgebungen schnell genutzt werden kann.
- Klare Warnung auf der Seite, dass die Datei nach erfolgreichem Lauf gelöscht werden muss.

## Sicherheit

- Die Datei ist bewusst öffentlich erreichbar und daher nur als temporäres Wartungswerkzeug gedacht.
- Nach Ausführung muss `public/migration.php` vom Server gelöscht werden.
- Die App selbst bleibt unverändert rollen- und CSRF-geschützt.

## Datenbank

- Keine neue Migration.
- Das Tool führt nur bestehende Migrationen aus.

## Tests

- PHP-Syntaxprüfung für `public/migration.php`.
- ZIP-Prüfung.
